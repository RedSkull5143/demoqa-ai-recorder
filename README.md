# DemoQA AI Recorder

An intelligent test automation system that records manual browser interactions 
and converts them into framework-ready Java Selenium TestNG test cases using AI.

## Vision
Record once → AI understands → Get production-ready test code in your framework style.

## Tech Stack
- Java + Selenium + TestNG (framework)
- Python + Groq (AI layer)
- ChromaDB (vector store)
- LLaMA 3.1 (LLM)

## Project Phases
- [x] Phase 1 — Base framework (DemoQA)
- [x] Phase 2 — Browser action recorder
- [ ] Phase 3 — AI mapper (recorded actions → framework methods)
- [ ] Phase 4 — Code generator (TestNG class + JSON output)

## Setup
1. Clone repo
2. Run `mvn compile`
3. Run `mvn test`
